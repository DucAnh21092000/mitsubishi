Tìm hiểu HTML, CSS, Javascript Để có thể lập trình web PHP FullStack (nôm na là lập trình “cả cái web PHP”) thì đầu tiên người học phải tìm hiểu và biết cách lập trình HTML, CSS, Javascript – gọi là lập trình web Front-End (Sau lập trình Front-End sẽ tiếp tục tìm hiểu cách lập trình web Back-End bao gồm lập trình chức năng web bằng PHP, CSDL SQL v.v. sẽ được giới thiệu trong các bài sau). Trong bài này chúng ta sẽ tìm hiểu tổng quan về lập trình web Front-End bao gồm HTML, CSS, Javascript và tổng hợp tài liệu hướng dẫn để tự học lập trình các phần này. HTML – Tìm hiểu và thực hành lập trình Câu hỏi: HTML là gì? Trả lời: HTML là một dạng ngôn ngữ đánh dấu bằng các thẻ <>, HTML thường được dùng để tạo giao diện các trang Web. Trình duyệt Web (Web Browser) “đọc” HTML của trang để hiển thị giao diện trang Web. HTML bao gồm rất nhiều phần tử (elements) để cấu trúc giao diện một trang web, các phần tử HTML được thể hiện bằng các thẻ (tags) ví dụ như “đoạn” , “tiêu đề” , “nội dung” v.v. HTML cần được học khi nào? Khi bạn muốn học lập trình web nền tảng bất kỳ (như PHP, Java, ASP) thì HTML là một trong số những phần cần được học đầu tiên, vì sao? Vì xây dựng giao diện web cần phải sử dụng HTML, mà gần như trang web nào cũng có giao diện vì vậy cần học HTML trước. Tự học HTML ở đâu? Trả lời: Bạn có thể tự học HTML bằng cách tìm hiểu và thực hành cách sử dụng HTML ở các trang nổi tiếng sau: https://www.w3schools.com/html/default.asp https://www.tutorialspoint.com/html/index.htm
